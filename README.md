## About project

Simple application introduce list of air tickets with stops filter and currency changer (current course).

Technologies: React, TypeScript, MobX.

## Installation

```
npm install
```

#### Before commits

```
npm run prepare
```

### Run project

#### Dev

```
npm run start:dev
```

## Or run in docker

```
docker-compose up
```
